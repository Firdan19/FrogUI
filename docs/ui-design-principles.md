# UI Design Principles

FrogUI is built for one professional operator.

## Required Identity

- Luxury minimal.
- Matte color system.
- Borderless layout.
- Glass-overlay surfaces.
- Calm, dense, professional ergonomics.

## Excluded Patterns

The product must not include UI elements for:

- Joining a community.
- Social feeds.
- Public profiles.
- Invite flows.
- Team chat.
- Follower counts.
- Badges or gamified public status.

## Component Rules

- Web Components are the source of truth.
- Shadow DOM isolates component styling.
- React wrappers must not fork visual behavior.
- Layouts should feel like a private cockpit, not a marketing page.
